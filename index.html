Working Paper

Discovering a Measurement Artifact in Polymarket's lastTradePrice

Analyzed 76,000+ resolved Polymarket contracts and identified a systematic lastTradePrice measurement artifact — a structural bias in how final prices are recorded. Exploiting this artifact, I built a contrarian mean-reversion strategy achieving a Sharpe ratio of 2.37, outperforming naive market benchmarks. The pipeline spans data collection, artifact characterization, backtesting, and out-of-sample validation.

AI Agent Eval

Stress-Testing AI Agents on Silent Data-Science Bugs

Designed and built an 8-task evaluation suite measuring whether AI coding agents can detect, diagnose, and fix silent data-quality and ML-methodology bugs — pipelines that run without error yet report inflated metrics (target leakage, temporal leakage, join inflation, schema ambiguity). Each task stacks multiple defects behind deterministic, seeded verifiers. I ran ≥3 trials per task against gemini-3.5-flash through the Harbor / terminus-2 harness, with a full failure analysis grounded in agent-eval literature (DAB, MMTU, AutoDCWorkflow, Shankar et al.).
